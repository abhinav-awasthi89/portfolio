LLM-Powered AI Agent

I developed an AI agent leveraging smolagents, integrating LLM-powered decision-making with external function calls to enhance its capability to interact with various tools dynamically. To optimize its functionality, I designed and incorporated custom tools tailored for specific tasks, including data retrieval, grammar correction, and web search. These tools enable the agent to process diverse user queries efficiently, ensuring accurate responses and improved user experience. The backbone of this system is Qwen2.5-Coder-32B-Instruct, a powerful language model that interprets user inputs, determines the most relevant actions, and seamlessly triggers the appropriate tools. This integration allows the AI agent to execute complex workflows while maintaining adaptability and efficiency in real-time interactions.

PYTHON, LLM, Tool-agents

Admission Prediction System

I developed a predictive model for graduate admissions using linear regression, achieving an impressive 90% accuracy in predicting admission probabilities based on applicant profiles. To enhance model performance and interpretability, I implemented feature engineering and data preprocessing techniques, ensuring that the input data was optimized for effective learning. The project was built using the FARM stack (FastAPI, React, and MongoDB), enabling seamless integration between the machine learning model and backend API. This architecture allowed for efficient data handling, real-time predictions, and an interactive user experience, making the model both scalable and user-friendly.

PYTHON, FastAPI, ReactJS, MongoDB
